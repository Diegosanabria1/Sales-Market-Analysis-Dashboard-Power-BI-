# Sales-Market-Analysis-Dashboard-Power-BI-
Dashboard interactivo desarrollado en Power BI para el análisis de ventas del sector automotriz. Incluye indicadores clave (KPIs), análisis temporal de ingresos, desempeño por región y modelo, así como la evaluación de la adopción de vehículos eléctricos (BEV) en relación con variables económicas como el precio del combustible
